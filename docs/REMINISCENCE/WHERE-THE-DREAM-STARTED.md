影咖

------

很多事情在发生之前其实已有端倪。

很多时候我相信即使再来一遍，还是会一样的结果，该发生的总会发生，不会更早，也不会更迟。

我和她，亦如是。

&nbsp;

***2020.7.16 晚***

他缩在被窝里，握着微微发汗的手机。明天就是两人这个假期的约定见面——从某次开始成为他假期的保留节目。他隐隐觉得很兴奋，和她的每次见面总是很放松，什么都不用想，很快乐，是完完全全的玩耍。而且，这次母亲已然同意了，理由自然是老套的和高中那帮人一起出去——对此，她大约会翻一个意味深长的白眼。

剩下的问题是，他们去做什么呢？

看电影？似乎有点暧昧。也无所谓，反正她喜欢就好。时值疫情，影院都没怎么开。“我看到私人影院”，她提出了一个折衷的方案。一瞬间，知乎上私人影院里发生的各种故事如潮水般涌上记忆。没想到她会提出这个，虽然吧，私人影院确乎是个极佳的看电影的场所，就是在他心中气氛旖旎。这个年轻人倒是愿意成家以后自己有个影房，完全隔音，想看啥就看啥，想想就很过瘾。

“还是试探一下她的想法比较好”。屏幕上汗涔涔的，是的，这小子四季都汗手，记事起就是这样。而她倒是正好相反，以前牵上的时候两个人深有体会。抓住被子用力地擦了擦屏幕，又花了两秒地听了一下没有父母起夜的声音，他轻轻地打下：“用户评价：带大屏幕的宾馆”，Enter。

“哈哈哈”，她简单的回复道。看来，她并不介意。好极，那只要确定看什么电影就行。

挑挑拣拣，两人最后决定还是看点恐怖的，至于哪家店呢，其实不重要，他内心已经在期待明天会发生什么了。别看有的人表面上一本正经，其实背地里不知道会在想什么事情。很难讲为什么他那个时候会这么期待和他发生什么，大概，有些致命的吸引力不是这个人所信仰的物理和数学能够解答的。也许是他已经在自己的这方世界呆的足够久了，而她所展示的异世界又是那么丰富多彩，和他熟悉的冰冷黑白有着天壤之别。二十多年的浅薄经验也告诉不了他什么，干脆就不去想，甘心地做一片随波逐流的落叶，反正命运会告诉他答案的。

简单敲定事宜之后，他便满怀期待的晚安了。

&nbsp;

***2020.07.17 中午***

有时候，关键的事情总会在最不经意的时候发生。

见到她的时候已然是中午。那天的太阳不算大，至少记忆里是这样的。

他习惯性的扫了一眼就看向别处，对当时并没有留下深刻的印象，倒是仔细的深吸了一口气。直直地盯着一个女孩子看是不礼貌的，即使她总是那么有魅力。

简单的磋商，两个人决定就吃点十八家面馆的米面，比较清爽。

店面不大，两人一前一后上了楼，坐在窗边。对面的楼房只比他们矮一层，相隔一条窄窄的步行街，低头近的仿佛能摸到，抬头又是开阔的天际线。

上次见面大约是上个假期了吧，他这么想着，显得稍稍拘谨。倒是她斜斜地坐在高脚椅上，叽叽喳喳地吐槽起近况。她总是有那么多话要讲，眼前的男孩子觉得很是新奇。如果在他的心里对好友也有列表分组的话，那这个女孩子一定是独一组的。他们从近况聊到曾经的事情，从学习谈到其他人的八卦。有那么一瞬间，他觉得自己回到了初中，然后快速地经历了她的中考高考大学生活，曾经她讲过的欢乐往事和不愉快的经历像彩绸上流转的颜色交织在一起，他好像看到她笑过委屈过，坚毅的她沮丧的她交替在一起，最后变成了现在的她。

终于，他们的聊天差不多暂时告一段落，慢慢地下楼，店铺冷冷清清的中午打烊了。推开门，外面还是有点小小的太阳。

&nbsp;

出了这里，下一站理应是之前看好的影咖。可惜，缺少经验的他没有提前订好，满了。

经过小小的纠结和衡量，两个人决定去另一家影咖，稍远。七拐八拐，他们终于摸到了影咖。进门，“热死了”，她一把摘下自己的帽子，散开自己的头发，浑身散发着属于夏天的热汽。

仅剩的两个房间中，他是不喜沙发的，又没法下定决心开口要另一个。他在犹豫，在等她表态。

她从角落里舒服的椅子上站起来，凑上来翻了翻，表示也不喜欢沙发的。

结束。他涌上一种难以置信的感觉，事情正在不可阻挡地向可能的方向发展，会怎么样呢。

&nbsp;

进了房间，不大不小的房间里铺着一个床垫一样的床，前面是银幕，看起来相当不错。

两个人盘腿坐下了，关上房门，调暗灯光。

他感觉自己的心跳有点快。

“看啥咧，你有什么想看的么”，他翻起电影，“恐怖片？剧情片？科幻片？还是动漫？”。他有意避开爱情片，感觉似乎不适合。好吧，看起来她并没有太多的想法，只好任男生逐一介绍自己知道的电影。“这个我看过，讲的是··· ···”，“这个我知道，是关于··· ···”，“这个，我看过剧情解说，还可以吧，是··· ···。你想看下么？”

十分钟后。

还是没有结果，没错，两个重度选择困难症，是很难决定看什么的。

时至今日，他已经不记得是谁提出看《她》的了，他只记得当时自己说的是——这是讲和AI谈恋爱的，剧情片。

&nbsp;

灯幕暗下，电影开始。

他和她盖的是同一条被子。还好，他心想，温度正好，就是奇怪了点。

一边看着，一边给她解说前面他知道的剧情。其实，他明明知道大部分的，是和新疆室友一起看的，当时的他并没有很深刻的感受，就是对结局有些唏嘘，以及，记住了寡姐的嗓音。

很快，到了男主文爱的片段。啊好吧没事她是个医学生，他这么想着，似乎也没有感受到她有多尴尬，就渐渐放松下来，仅仅是希望门口的人不要觉得里面在看什么奇怪的电影就好。

&nbsp;

电影过半，他也有点心不在焉起来，毕竟是看过的电影，他可以毫不费力的一边看一边解说聊天，一边想点别的。也许是气氛，也许是他早有此意。

一硝二硫三木炭，可燃物靠在一起总是要出事的。

她转过来了，半撑着身子。

今天的她是那么美，似笑非笑，眨巴着眼睛，薄薄的黑色披肩挂下来，在银幕的衬托下光影流转。黑色的长发像瀑布一样从柔软的脖颈后面放出来，遮住了他的其他思绪。他闻到了头发，是很好闻的，就像上次分别时他盯着她的背影想抱却不敢抱的味道。有一点点香香的，如果说初中课本里的那滴香水是青涩的，那这回他无法描述的香氛就是成熟的感觉。

她说话了，好近。

他盯着她的眼睛也许看了好几秒，呼吸慢得自己都察觉不到，然后是鼻子，衬在小小的没有岁月痕迹的小脸上，然后是有点翘的嘴唇。

他想亲她。

就是现在。

印上去了。

闭着眼睛，好软，真的好软。他只剩下这一个念头。无法满足的真实满足感直扑心脏，然后被迅速的泵向全身。他想记住这一刻的感觉，但是它又像最清的溪水流过指间，只剩一点潮湿提醒他真实存在的过去。

&nbsp;

打火机是不会无缘无故冒出火苗的，除非有人已经压在扳机上很久了。

分开的两人在黑暗中对视，他想从她眼中读出点什么。

她动了，他闭眼了。

伸手把她抱住，这一刻他等很久了。对她的需要、信任、渴望在这一刻通过最亲密的接触正在毫无保留的倾泻出去。

她的舌尖在门口徘徊试探。来吧来吧，他情愿融化在她的曲线上，带着可说的不可说的一切所有。这是这么多年来他从未体验过的感觉，想要把自己交出去，任她探索。他想要贴得再紧一些，再契合一些，感受她。命运的红线从很早时候就开始纠缠，近过，也远过。而这一次，却比以往来得更加猛烈。

他透了一口气，把自己的脖子嵌在她舒服的脖颈里，两人交颈相拥。

没有人说话，房间里似乎也没有电影在放。他的世界里只剩下眼前的女孩子，和拥抱的感觉。

她轻轻地拍了拍他的背，似乎比七年前的拥抱更多了一些意味。

他不是七年前的小孩子，恐怕她也不再是七年前的小女孩，而是一个大姑娘了，散发着该死的魅力。她有种让他卸下一切的冲动，自己却单纯还是像七年前的样子。

&nbsp;

打破耳鬓厮磨的是服务员的敲门，提醒时间快到了。

有的人，进门的时候还是肩并肩，走的时候就成了手牵手。

太阳还是敬业的挂在天上，那是夏天下午的太阳，他们以前几乎没有这么从阳光底下走过。他对两个人记忆最深的还是课上的牵手和放学回家前在角落里的拥抱，那次的漫展也许是少见的一起出去玩的经历。

压着马路，沐浴着这个夏天难得的阳光，这将是一个新的开始。

&nbsp;

那天晚上在去茶去，两个人紧紧地腻在桌子一边的沙发上，腻了大半个晚上。说不尽的话，亲不完的吻，旁边的情侣和他们比起来都显得像是刚相亲的对象。

火柴刚点着的时候总是最猛烈的，第一口可乐总是最好喝的，第一口嗑的药总是最上头的。他的脑子今晚已经完全的被她占据了，每条褶皱里都透出数不清的粉色泡泡。

一个完全因为爱情断片的人，那天最后只记得将她吻别后迷离地回家。靠在出租车的玻璃上，外面划过的流光像是不需要许愿的流星划过天空，有的事物本不在他的期望里，也奢侈的令人发指。

&nbsp;

***2020.07.17 23:50***

“所以 你是我的了嘛现在”

“可以吧”

“我需要一个比较确切的答案”

“可以”

&nbsp;

就这样，他们又回到了原点。有的人其实还没准备好，虽然他已经压在扳机上很久了。不过，不管这回是化为灰烬还是能陪他度过漫漫长夜，他都想倾尽全力去爱她。
